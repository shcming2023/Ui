# 快速开始指南 - AI自动标注功能

## 🚀 5分钟快速上手

### 第一步：定义你的分类体系（3分钟）

1. **打开元数据管理页面**
   - 点击左侧导航栏的"元数据管理"

2. **配置结构化字段**
   - 点击"结构化字段" Tab
   - 查看预设的9个字段（学科、学段、课程标准等）
   - 根据需要点击"添加字段"创建自定义字段
   - 点击字段卡片右侧的"编辑"按钮修改预设值

3. **创建灵活标签**
   - 点击"灵活标签" Tab
   - 点击"创建标签"按钮
   - 输入标签名称和选择分类
   - 系统已预设了常用标签（数学、高考、阅读理解等）

### 第二步：配置AI自动标注规则（2分钟）

1. **打开AI规则配置**
   - 在元数据管理页面点击"AI自动标注规则" Tab

2. **查看预设规则**
   - 系统已预设5个常用规则，默认全部启用
   - 每个规则显示：触发条件、执行动作、成功率

3. **调整规则（可选）**
   - 使用Toggle开关启用/禁用特定规则
   - 点击"设置"图标调整规则参数
   - 点击"编辑"修改规则条件

4. **配置执行设置**
   - 确认"上传时自动执行"已勾选（推荐）
   - 根据需要调整置信度阈值（默认85%）

### 第三步：上传资料并体验AI分析（30秒）

1. **打开原始资料库**
   - 点击左侧导航栏的"原始资料库"

2. **上传文件**
   - 点击右上角"上传资料"按钮
   - 拖拽或选择PDF/DOC/PPT文件
   - 确认"上传后自动启动AI智能分析"已勾选 ✓
   - 点击"开始上传"

3. **查看AI分析结果**
   - 上传完成后，资料卡片显示"AI分析中..."（紫色脉冲）
   - 几秒钟后显示"AI已分析"（绿色徽章）
   - 查看AI识别的元数据和标签（紫色背景框）

## 📖 常见使用场景

### 场景1：批量上传高考真题

**需求：** 我有50份不同省份的高考数学真题PDF，需要快速整理入库

**操作步骤：**
```
1. 确认"自动识别学科"、"考试类型识别"规则已启用
2. 批量上传所有PDF文件
3. 系统自动识别每份试卷的学科、年份、地区
4. 自动打上"数学"、"高考"、"真题"标签
5. 在筛选面板选择"数学"+"高考"快速查找
```

**预期效果：**
- ⏱️ 节省时间：从手动标注50×2分钟=100分钟 → 自动5分钟
- ✅ 准确率：98%（考试类型识别规则）
- 🏷️ 自动标签：每份资料自动打上3-5个标签

- 📚 便捷检索：按知识点快速查找相关资料

## ❓ 常见问题

### Q1: AI分析需要多长时间？
**A:**
- 单个PDF（10页）：5-10秒
- 批量10个文件：30-60秒（并行处理）
- 显示"AI分析中"时为实时状态

### Q2: AI识别错误怎么办？
**A:**
- 点击资料卡片进入详情页
- 手动编辑元数据和标签
- 系统会记录修正，用于优化模型

### Q3: 可以修改AI已标注的结果吗？
**A:**
- 可以，所有AI结果都可以手动修改
- 在资料详情页点击"编辑"按钮
- 或使用"批量打标签"覆盖现有标签

### Q4: 如何禁用某个规则？
**A:**
- 进入"元数据管理" → "AI自动标注规则"
- 找到目标规则，点击Toggle开关
- 灰色=禁用，蓝色=启用

### Q5: 规则会影响已上传的资料吗？
**A:**
- 不会，规则只对新上传或手动触发分析的资料生效
- 如需对已上传资料应用新规则，请：
  - 选择目标资料
  - 点击"批量AI分析"重新分析

### Q6: 能否导出/导入规则配置？
**A:**
- 当前版本暂不支持（计划中的功能）
- 可在元数据管理页面查看所有规则配置
- 建议截图保存规则设置备份

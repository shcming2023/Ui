# 教育内容资产管理平台 (EduAsset CMS)

一个专业的B端教育内容全生命周期管理系统。

## 🎯 产品特点

### 完整的内容生命周期管理
从原始教育资料到最终成品的完整流程追踪和管理：
- **原始资料** - PDF、DOC、PPT等各类教育文档
- **Rawcode** - 机器解析后的中间产物（MinerU输出）
- **Cleancode** - AI清洗后的标准化内容
- **成品** - 题库、讲义、试卷、练习册、课程包

### 核心功能

✨ **血缘关系追踪** - 完整的资产血缘关系可视化，可从成品回溯到原始资料

📊 **元数据管理** - 结构化字段 + 灵活标签双体系，支持自动抽取

🔄 **流程可视化** - 清晰的任务流程、进度追踪和状态管理

👁️ **权限分层** - 内部可见/生产可见/审核可见/对外可见四级权限

📈 **统计分析** - 资产使用情况、标签热度、任务统计

🔍 **高级筛选** - 多维度筛选和搜索，快速定位资产

## 🎨 设计系统

### 颜色方案
- **主色：** 蓝色系（专业、科技感）
- **辅助色：** 橙色（处理中）、绿色（完成）、红色（失败）、黄色（待审核）
- **中性色：** 深蓝侧边栏、灰白背景、slate系文字

### 组件规范
- **卡片** - 圆角、边框、悬停阴影
- **徽章** - 状态标识，颜色编码
- **标签** - 圆角小标签，支持多色
- **表格/列表** - 清晰的数据展示
- **时间线** - 流程和历史展示

## 💡 使用场景

### 内容入库
1. 批量上传原始教育资料
2. 自动抽取元数据和打标签
3. 分类归档到资料库

### 解析处理
1. 选择原始资料发起解析
2. MinerU 生成 rawcode
3. AI 清洗生成 cleancode
4. 人工审核和修订

### 成品生成
1. 从 cleancode 选择内容
2. 配置成品类型和参数
3. 自动生成题库/讲义/试卷
4. 审核后发布到成品库

### 追溯查询
- 从成品反查来源 cleancode
- 查看完整处理链路
- 分析内容使用情况

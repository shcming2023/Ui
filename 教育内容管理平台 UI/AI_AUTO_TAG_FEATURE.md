# AI自动标注功能 - 详细说明

## 🎯 功能概述

教育内容资产管理平台的核心功能之一，允许用户自定义分类和标签体系，并通过AI自动识别和标注上传的教育资料。

---

## 📋 核心功能

### 1. 用户自定义分类和标签体系

#### 1.1 结构化字段管理
在**元数据管理页面** > **结构化字段** Tab中：

- **预定义字段类型：**
  - 学科（数学、语文、英语、物理、化学等）
  - 学段/年级（小学、初中、高中、具体年级）
  - 课程标准（新课标2022、义务教育课标、人教版、部编版等）
  - 地区（全国、各省市、国际地区）
  - 语言（中文、English）
  - 资料类型（教科书、练习册、试卷、讲义、课件、题库）
  - 处理阶段（原始资料、rawcode、cleancode、成品）
  - 权限级别（内部可见、生产可见、审核可见、对外可见）

- **支持操作：**
  - 添加新字段
  - 编辑字段值
  - 删除字段
  - 设置字段类型（单选/多选）

#### 1.2 灵活标签管理
在**元数据管理页面** > **灵活标签** Tab中：

- **标签分类：**
  - 课程标准
  - 学科
  - 考试（高考、中考）
  - 学段
  - 题型（阅读理解、几何、语法等）
  - 知识点
  - 教材版本

- **标签功能：**
  - 创建标签
  - 编辑标签
  - 删除标签
  - 查看标签使用统计
  - 按分类筛选标签
  - 查看使用趋势

- **智能建议：**
  - 新建标签建议（基于内容频繁出现的词）
  - 标签合并建议（检测重复或相似标签）

### 2. AI自动标注规则配置

在**元数据管理页面** > **AI自动标注规则** Tab中：

#### 2.1 预设规则
系统提供5种预设规则：

1. **自动识别学科**
   - 触发条件：文件内容包含学科关键词
   - 执行动作：自动添加学科标签
   - 成功率：96%

2. **课程标准识别**
   - 触发条件：标题或内容包含"新课标"、"人教版"等
   - 执行动作：自动添加课程标准标签
   - 成功率：94%

3. **考试类型识别**
   - 触发条件：标题包含"高考"、"中考"、"真题"
   - 执行动作：自动添加考试类型标签
   - 成功率：98%

4. **年级智能判断**
   - 触发条件：AI分析内容难度和知识点
   - 执行动作：自动赋值年级字段
   - 成功率：91%

5. **资料类型分类**
   - 触发条件：根据文件结构和内容特征
   - 执行动作：自动分类为试卷、练习册、讲义等
   - 成功率：87%

#### 2.2 规则管理功能

- **启用/禁用规则：** 一键开关特定规则
- **调整优先级：** 控制规则执行顺序
- **编辑规则：** 修改触发条件和执行动作
- **删除规则：** 移除不需要的规则
- **新建规则：** 创建自定义AI标注规则
- **批量测试：** 在实际应用前测试规则效果

#### 2.3 执行设置

- **上传时自动执行：** 资料上传完成后立即触发AI分析
- **并行执行规则：** 多个规则同时执行以提高速度
- **人工审核确认：** AI标注结果需人工审核后生效
- **低置信度提醒：** 当AI置信度低于阈值时发送通知
- **置信度阈值设置：** 可调节（默认85%）

#### 2.4 统计和监控

- **规则统计：**
  - 总规则数
  - 已启用/已禁用规则数
  - 总执行次数
  - 平均成功率

- **最近执行记录：**
  - 执行的规则名称
  - 执行时间
  - 执行状态（成功/警告）
  - AI置信度

### 3. 资料上传流程中的AI应用

在**原始资料库** > **上传资料**弹窗中：

#### 3.1 上传选项

- **一键启用AI分析：**
  - 默认勾选"上传后自动启动AI智能分析"
  - AI将自动完成：
    - 识别文件内容和教育领域
    - 提取关键元数据
    - 根据预设规则自动打标签
    - 生成资料摘要和知识点结构

- **手动填写基础信息（可选）：**
  - 学科
  - 学段/年级
  - 资料类型
  - 课程标准
  - 注：AI分析后会自动填充或修正这些信息

#### 3.2 AI分析状态展示

资料卡片上显示AI分析状态：

- **AI分析中：** 紫色脉冲动画徽章
- **AI已分析：** 绿色徽章 + AI识别结果展示
- **待分析：** 黄色徽章 + 提示信息

#### 3.3 AI识别结果展示

已分析的资料会显示：
- AI识别到的元数据（学科、年级、类型、课程标准）
- 自动添加的标签
- 以特殊样式区分（紫色AI图标标识）

### 4. 批量操作

在**原始资料库**主页面：

#### 4.1 批量AI分析
- 选择多个未分析的资料
- 点击"批量AI分析"
- 系统按规则自动处理所有选中资料

#### 4.2 批量打标签
- 选择多个资料
- 打开批量标签弹窗
- 手动添加标签或选择常用标签
- 可链接到元数据管理配置自动规则

## 🔄 完整工作流程

### 流程1：首次上传资料（自动流程）

```
1. 用户上传PDF文件
   ↓
2. 勾选"上传后自动启动AI智能分析"（默认启用）
   ↓
3. 上传完成后，系统自动触发AI分析
   ↓
4. AI按优先级执行所有已启用规则：
   - 识别学科 → 添加"数学"标签
   - 识别课程标准 → 添加"新课标2022"标签
   - 识别考试类型 → 添加"高考"标签
   - 判断年级 → 设置"高三"字段
   - 识别资料类型 → 设置"真题"字段
   ↓
5. 资料卡片显示"AI已分析"徽章
   ↓
6. 显示AI识别的元数据和标签
```

### 流程2：手动触发AI分析

```
1. 资料已上传但未分析（显示"待分析"徽章）
   ↓
2. 点击资料卡片上的"启动AI自动分析"按钮
   ↓
3. 系统执行AI分析流程
   ↓
4. 完成后显示识别结果
```

### 流程3：配置自定义标注规则

```
1. 访问"元数据管理" → "AI自动标注规则" Tab
   ↓
2. 点击"新建规则"
   ↓
3. 配置规则：
   - 规则名称
   - 触发条件（关键词、内容特征等）
   - 执行动作（添加标签、设置字段）
   - 优先级
   ↓
4. 启用规则
   ↓
5. 后续上传的资料自动应用此规则
```

## 💡 使用场景示例

### 场景1：批量上传高考真题
**需求：** 上传100份不同年份的高考真题

**解决方案：**
1. 配置AI规则识别"高考"关键词
2. 批量上传所有PDF
3. AI自动识别每份资料的学科、年份、地区
4. 自动打上"高考"、"真题"、对应学科和年份标签
5. 无需人工逐个标注

### 场景2：区分不同版本教材
**需求：** 管理人教版、部编版、苏教版等不同版本的教材

**解决方案：**
1. 在结构化字段中配置"课程标准"字段
2. 设置AI规则识别标题和内容中的版本信息
3. 上传时AI自动识别并标注版本
4. 筛选时可按版本快速查找

### 场景3：知识点自动提取
**需求：** 从大量练习册中提取知识点标签

**解决方案：**
1. 创建灵活标签：几何、代数、概率统计等
2. 配置AI规则识别数学知识点关键词
3. 上传练习册后AI自动提取知识点
4. 后续可按知识点组织题库和资料

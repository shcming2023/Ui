## 🎯 核心工作流程

### 流程1：用户定义知识库分类和标签

1. 访问"元数据管理"页面
2. 在"结构化字段" Tab添加/编辑预定义字段
3. 在"灵活标签" Tab创建自定义标签
4. 在"AI自动标注规则" Tab配置识别规则

### 流程2：AI自动标注材料

**方式A：上传时自动**
1. 上传资料，勾选"自动启动AI分析"
2. 上传完成后，AI自动执行所有已启用规则
3. 资料卡片显示AI识别结果

**方式B：手动触发**
1. 已上传但未分析的资料显示"待分析"徽章
2. 点击"启动AI自动分析"按钮
3. AI执行分析并展示结果

**方式C：批量处理**
1. 选择多个待分析资料
2. 点击"批量AI分析"
3. 系统批量处理所有选中资料

## 📊 数据流示意

```
用户上传资料
    ↓
[可选] 勾选"AI自动分析"
    ↓
文件上传完成
    ↓
触发AI分析流程
    ↓
按优先级执行所有已启用规则：
├─ 规则1: 识别学科 → 添加学科标签
├─ 规则2: 识别课程标准 → 添加标准标签
├─ 规则3: 识别考试类型 → 添加考试标签
├─ 规则4: 判断年级 → 设置年级字段
└─ 规则5: 分类资料类型 → 设置类型字段
    ↓
记录执行结果（成功/失败，置信度）
    ↓
更新资料元数据和标签
    ↓
UI显示"AI已分析"徽章和识别结果
```

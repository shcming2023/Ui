# 教育内容资产管理平台 - 系统概览

## 产品定位
覆盖教育内容全生命周期的内部B端管理平台，统一管理从原始资料到最终成品的完整流程。

## 核心流程
```
原始资料 → Rawcode → Cleancode → 成品
```

## 系统架构

### 主要页面

1. **工作台 (Dashboard)** - `/`
   - 统计概览：原始资料数、处理中任务、cleancode数量、成品数量
   - 最近入库资料列表
   - 待处理任务进度
   - 最近生成成品展示
   - 热门标签云

2. **原始资料库** - `/source-materials`
   - 左侧高级筛选面板（学科、学段、格式、状态、课程标准）
   - 资料卡片网格展示
   - 批量导入、批量打标签、发起解析功能
   - 支持搜索和排序

3. **资产详情页** - `/asset/:id`
   - 基础信息和元数据展示
   - 文件预览区
   - 血缘关系可视化（原始→rawcode→cleancode→成品）
   - 处理历史时间线
   - 版本管理
   - 权限设置
   - 相关资产推荐

4. **处理中心** - `/process-workbench`
   - 任务状态概览（待处理、处理中、待审核、失败）
   - 任务列表与进度追踪
   - 任务详情：输入输出、负责人、执行时间
   - 失败任务错误日志
   - Rawcode vs Cleancode 对比功能

5. **成品库** - `/products`
   - 成品卡片展示（题库、练习册、试卷、讲义、课程包）
   - 按类型、学科、评分筛选
   - 使用统计和评分展示
   - 回溯来源功能

6. **元数据管理** - `/metadata`
   - 结构化字段配置（学科、学段、课程标准等）
   - 灵活标签管理
   - 标签统计和使用趋势
   - 自动抽取建议

7. **任务中心/审核台** - `/tasks`
   - 全部任务监控
   - 按状态筛选（待处理、处理中、待审核、失败、已完成）
   - 任务执行日志
   - 审核功能（通过/驳回）
   - 失败任务重跑

## 核心对象

### Asset（内容资产）
- 原始资料（PDF、DOC、PPT、扫描件）
- Rawcode（MinerU输出的markdown、latex、图片）
- Cleancode（清洗后的标准化内容）
- Product（最终教育成品）

### Metadata（元数据）
**结构化字段：**
- 学科、学段、年级、课程标准
- 地区、语言、资料类型
- 处理阶段、权限级别

**灵活标签：**
- 新课标2022、人教版、部编版
- 高考、中考、专项练习
- 阅读理解、几何、语法等

### Process Job（处理任务）
- MinerU解析
- AI清洗
- OCR识别
- 格式转换
- 成品生成

### Version（版本）
- 版本号、状态、创建时间
- 版本历史和回滚

## 使用场景

1. **内容入库流程**
   - 上传原始资料 → 自动元数据抽取 → 分类打标签

2. **解析处理流程**
   - 发起MinerU解析 → 生成rawcode → AI清洗 → 生成cleancode

3. **成品生成流程**
   - 选择cleancode → 配置成品类型 → 生成题库/讲义/试卷

4. **审核发布流程**
   - 人工审核 → 修订调整 → 审核通过 → 入成品库

5. **追溯查询**
   - 从成品反查来源cleancode和原始资料
   - 查看完整处理历史和血缘关系

## 权限体系
- 内部可见：团队内部查看
- 生产可见：生产人员可操作
- 审核可见：审核人员可审批
- 对外可见：可用于对外服务

---

**设计目标：** 提供清晰、专业、高效的B端内容资产管理体验，支持教育内容从采集到成品的全生命周期管理。
